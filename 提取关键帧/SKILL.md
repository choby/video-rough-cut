# 提取关键帧

> 从“转录.md“中提取关键帧

## 快速使用

```
用户: 帮我提取关键帧
用户: 提取关键帧
```

## 流程

```
根据用户输入的内容，在“转录.md“中提取包含用户输入内容的时间戳区间，将最后提取出的时间戳列表按原格式生成“片段{x}.md”文件
```

## 提取规则
- 无需总结
- 忽略与用户输入内容无关的语句
- 提取结果中时间区间内的语句完整输出
- 单次对话提取的所有内容按时间戳顺序保存在同一个片段文件中
- 忽略重复内容， 则只选取时间戳区间最连贯的部分

## 其他说明
其中 x 从 1 开始递增， 如片段1.md, 片段2.md, 片段3.md
